# daum.pw

个人主页与数字客厅。

## 技术栈

- 原生 HTML / CSS / JavaScript
- GSAP 动画库
- Node.js 轻量服务端

## 本地运行

```bash
node server.js
```

默认端口 `3456`，打开 http://localhost:3456 即可。
